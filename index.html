CBS — CONFLICT-BASED SEARCH

SPACETIME COLLISION ELIMINATION

CBS builds a conflict tree over all fleet paths simultaneously. When a spacetime conflict is detected (two trucks at the same cell at the same time), CBS inserts a wait constraint at the precise waypoint — resolving it without replanning the entire system. Only affected trucks are replanned. P2P GPS broadcasts every second keep all trucks synchronized. Result: conflict-free paths with zero deadlock.

HOW IT WORKS

The camera on each truck captures the dump pile from the side. CLAHE (Contrast Limited Adaptive Histogram Equalization) enhances the image, and a gradient-based pseudo-depth map estimates the pile's 3D profile in real time.

TILT ANGLE FORMULA
θ = arctan( (D_bottom − D_top) / H )
D_bottom = depth at pile base  ·  D_top = depth at pile peak  ·  H = pile height in pixels
WHAT THE ANGLE TELLS US
θ < −10°  (tilt LEFT) → Dump RIGHT-of-center
θ > +10°  (tilt RIGHT) → Dump LEFT-of-center
|θ| < 10°  (balanced) → Dump centered ✓
FILL RATIO
Fill Ratio = D_corrected / D_max_capacity
D_corrected accounts for tilt offset. When Fill Ratio → 1.0, spot marked FULL.
